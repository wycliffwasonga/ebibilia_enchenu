# Ebibilia Enchenu

Ebibilia Enchenu is a [Ekegusii](https://en.wikipedia.org/wiki/Gusii_language) Bible translation. This project aims to get this translation available beyond just the print media i.e. into Bible study softwares available today, web and etc.

This project uses [USFM](https://ubsicap.github.io/usfm/) format.

Note: This is an ongoing project and currently it is very incomplete.
